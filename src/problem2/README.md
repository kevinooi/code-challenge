## âœ¨ Features

- **Modern Tech Stack**: Powered by [Vite](https://vite.dev/) + [React](https://react.dev/).
- **Premium Aesthetics**: Glassmorphism design system built with [Tailwind CSS](https://tailwindcss.com/).
- **Seamless Animations**: Smooth modal transitions and micro-interactions using [Framer Motion](https://www.framer.com/motion/).
- **Live Market Data**: Mimics real-time token prices from the [Switcheo API](https://interview.switcheo.com/prices.json) with polling every 60 seconds.
- **Bidirectional Sync**: Intuitively calculates exchange amounts as you type in either field.
- **Smart Search**: Searchable token selector with the currently active token pinned to the top.
- **Validation**: Includes balance checks, input sanitization, and simulated on-chain transaction states.
